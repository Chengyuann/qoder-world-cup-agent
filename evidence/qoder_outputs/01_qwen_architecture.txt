## WorldScope 世界杯冠军预测 Agent — 技术总结

### 一、系统架构

本项目采用**纯前端单页应用**架构，基于 React 18 + TypeScript 5.7 + Vite 6 构建，零后端依赖，所有计算均在浏览器内完成。整体分为四层：

- **数据层（Data Collector）**：内嵌 48 支球队的完整数据集，涵盖 FIFA 排名、综合实力、近期状态、阵容深度、淘汰赛经验、差旅适应、攻防能力等 8 项指标，按 2026 赛制分为 12 个小组。
- **推理层（Reasoning Core）**：确定性加权评分引擎，由复合评分、Logistic 胜率模型、期望进球模型和 Softmax 概率分布四个模块串联组成。
- **展示层（Visual Layer）**：深色主题响应式 UI，包含英雄区、权重控制面板、概率排行榜、淘汰赛对阵图、小组赛积分榜和架构说明六大板块。
- **部署层（Deploy Surface）**：通过 GitHub Actions 自动构建并部署至 GitHub Pages，支持一键访问。

### 二、数据流

数据流遵循单向管道模式：

**球队数据 + 用户权重 → 复合评分 → 小组赛模拟 → 淘汰赛推演 → Softmax 冠军概率 → UI 渲染**

用户通过 5 个滑块调整实力、状态、深度、经验、差旅五项权重（总和 100），每次调整后 `useMemo` 触发全链路重新计算，实时刷新所有预测结果，实现"调参即见效"的交互体验。

### 三、推理链路

推理引擎由四个阶段组成：

**1）复合评分**：对用户选定的五项权重做加权平均，得到每支球队的综合得分。

**2）比赛胜率**：基于 Logistic 函数，输入双方复合得分差与攻防错配值（主队进攻 vs 客队防守、反之亦然），经系数缩放后映射为胜率。系数 0.16/0.055/0.035 经过调校，使常见分差映射到合理概率区间。

**3）小组赛与淘汰赛模拟**：12 组各 6 场循环赛按积分、净胜球、复合得分排序，前两名直接晋级，最优 4 支第三名进入 32 强。淘汰赛采用硬编码对阵模板，逐轮推演至决赛，每场比赛自动生成中文三要素解读（实力对比、阵容深度、大赛经验）。

**4）冠军概率分布**：在复合评分基础上叠加攻防加成与大洲修正（欧足联/南美足联 +1.8），经温度 4.3 的 Softmax 归一化为 48 队的相对概率分布。默认权重下，**法国队**以最高概率夺冠，阿根廷、西班牙、巴西、德国紧随其后。

整条链路完全确定性、可复现，无随机采样，便于评审审计每一个推理环节。

### 四、可视化亮点

- **可解释权重面板**：5 个滑块对应 5 项因子，附带中文说明，用户可验证模型鲁棒性——拖动任一滑块，冠军归属与概率分布即刻响应。
- **淘汰赛对阵树**：从 32 强到决赛的完整 bracket，每张卡片展示预测比分、胜率及 AI 解读，决赛卡片额外展开三段式推理说明。
- **概率排行榜**：Top-10 球队以卡片+横向进度条呈现，直观展示概率梯度。
- **小组积分表**：12 组 4 队积分一览，晋级队绿色高亮、第三名琥珀色标注。
- **深色玻璃拟态设计**：采用 CSS 自定义属性、径向渐变与 glass-morphism 效果，支持 1100px/720px 两档响应式断点，移动端体验友好。

### 五、技术要点

| 项目 | 选型 |
|---|---|
| 框架 | React 18 + TypeScript 5.7 |
| 构建 | Vite 6 |
| 部署 | GitHub Pages + GitHub Actions CI |
| 模型 | 加权评分 + Logistic 回归 + Softmax |
| 赛制 | 2026 扩军版：48 队 / 12 组 / 32 强淘汰赛 |
